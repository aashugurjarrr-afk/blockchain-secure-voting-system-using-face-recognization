Blockchain-Based Secure Voting System

A blockchain-based voting system designed to improve election security and transparency.

Features
Secure vote storage
Tamper-proof records
Decentralized architecture
Technologies Used
python
Blockchain Concepts
HTML/CSS/JavaScript

## How to Run (Setup on a New PC)

1. Install Python (Python 3.10 or later recommended).

2. Clone or download this repository:
   git clone https://github.com/your-username/blockchain-secure-voting-system.git

   OR

   Download the ZIP file and extract it.

4. Install required Python libraries:
   pip install -r requirements.txt

   If `requirements.txt` is not available, install libraries manually:
   pip install flask
   pip install hashlib
   pip install requests

5. Open terminal in the project directory.

6. Run the backend server:
   python app.py

   - OR visit:
     http://127.0.0.1:5
8. The Blockchain Secure Voting System should now run successfully.
